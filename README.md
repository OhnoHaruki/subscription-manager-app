# Subscription Manager App

サブスクリプションや固定費をまとめて管理するためのスマホアプリです。

毎月・毎年発生する支払いを見える化し、支払い忘れやクレジットカードの有効期限切れによる決済失敗を防ぐことを目的としています。

## アプリ概要

このアプリでは、動画配信サービス、音楽サービス、クラウドストレージ、保険、通信費など、定期的に発生する支払いを登録して管理できます。

サブスクごとの金額、支払い周期、支払い方法、分類タグを管理することで、月ごと・年ごとの固定費を把握しやすくします。

## 目的

- サブスクの支払い管理を楽にする
- クレジットカードの有効期限切れによる支払い失敗を防ぐ
- 解約や支払い方法変更をスムーズにする
- 毎月・毎年の固定費を把握し、家計管理に役立てる

## 主な機能（MVP）

### サブスク管理

- サブスクの登録
- サブスク情報の編集
- サブスクの削除
- サブスク一覧表示

### 支払い方法管理

- クレジットカードの登録
- 銀行口座の登録
- サブスクごとの支払い方法設定

### 分類・集計

- タグによるサブスク分類
- 月額合計の表示
- 年額合計の表示

## 将来的な機能

- Gmail連携によるサブスク検出
- クレジットカードの有効期限アラート
- 解約導線のワンタップ化
- Web対応
- 支払い履歴の管理
- サブスクの利用頻度メモ

## 開発方針

まずはMVPとして、サブスクを手動で登録し、一覧表示と金額集計ができる状態を目指します。

その後、通知機能やGmail連携などを追加し、支払い失敗の予防や解約手続きの効率化につなげていきます。

## セットアップ

### 前提条件

| ツール | 用途 | インストール方法 |
| --- | --- | --- |
| [asdf](https://asdf-vm.com/) | Flutter バージョン管理 | 公式ドキュメント参照 |
| [bun](https://bun.sh/) | Firebase CLI のインストール | `curl -fsSL https://bun.sh/install \| bash` |

### 手順

#### 1. Flutter のセットアップ

```bash
# asdf で Flutter をインストール（.tool-versions のバージョンが使われる）
asdf plugin add flutter
asdf install

# 依存パッケージのインストール
flutter pub get
```

#### 2. Firebase CLI・FlutterFire CLI のセットアップ

```bash
# Firebase CLI のインストール
bun install -g firebase-tools

# FlutterFire CLI のインストール
dart pub global activate flutterfire_cli

# Firebase にログイン
firebase login
```

#### 3. Firebase プロジェクトの接続

[Firebase Console](https://console.firebase.google.com/) でプロジェクトを作成し、Authentication と Firestore を有効化した後:

```bash
flutterfire configure
```

#### 4. アプリの起動

```bash
flutter run
```
